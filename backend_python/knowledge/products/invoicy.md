# Invoicy

**Tagline:** Drop an Invoice. Get Clean Data.

## What It Is

Invoicy automatically OCRs, pre-processes, classifies, and extracts clean structured data from any invoice — whether native PDFs, scanned layouts, or photographed receipts. Define custom automation schemas and process high-volume batches with full read-only security.

## Supported Formats

PDF, Scanned PDFs, PNG, JPG, TIFF

## Key Stats

- 7-Step Automated Processing Pipeline
- Any format extracted
- 0 manual template setup required

## Core Capabilities

**Document Ingestion**
Upload native PDFs, scanned PDFs, images (PNG, JPG, TIFF), and more. Enable scanned document extraction mode for low-quality or photographed invoices, and choose your quality mode — optimise for speed on high volumes or maximum accuracy on critical documents.

**AI-Powered OCR**
A multi-engine OCR pipeline corrects page orientation, removes scan noise, and normalises layouts before extraction begins — so even crumpled, rotated, or faded invoices yield clean, structured data.

**Auto Classification**
Invoicy identifies whether each uploaded file is an invoice, credit note, purchase order, or receipt — before extraction begins. Each document type is routed to its matching extraction pipeline automatically, with no manual sorting required.

**Extraction Engine**
Select from several extraction strategies per document type — rule-based, model-based, or hybrid — depending on your accuracy needs and document variability.

**Validation and Review**
Extracted fields are validated against your custom business rules. Low-confidence fields are flagged in a human review queue for one-click correction — so nothing leaves the pipeline with gaps or misreads.

**Automation Rules**
Set automation rules triggered by document type, vendor name, invoice amount, or any extracted field. Matched documents are processed and routed automatically. Credits are debited per document, giving you full cost transparency.

**Workspace — Batch Review and Consolidation**
Create named workspaces for a legal matter, vendor account, or monthly close and attach all related invoices. Invoicy performs batch review across the entire group and generates a single consolidated data table from the whole workspace.

**Agentic Chatbot**
Every account includes a built-in agentic AI chatbot with full access to your extracted invoice data. Ask "What did we spend with Vendor X in Q1?" or "List all invoices over 50,000 pending review" — and get instant, accurate answers drawn directly from your workspace data. No SQL needed.

## Export and Integrations

CSV, JSON, SAP, Oracle, NetSuite, QuickBooks, Zoho Books, REST API

## 7-Step Automated Workflow

1. Upload document (any format)
2. OCR and intelligent pre-processing
3. Automatic document classification
4. Structured data extraction
5. Rule-based validation
6. Human review queue (flagged fields only)
7. Export clean structured data
